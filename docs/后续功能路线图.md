# 后续功能路线图

## 第一阶段：验收稳定性

- 完善学生课表冲突校验，避免同一学生选择时间重叠课程。
- 增加操作日志，记录登录、新增、修改、停用、选课、退课和成绩录入。
- 补充功能模块图、业务流程图、ER 图和部署运行说明。

## 第二阶段：演示体验

- 增加学生、教师、班级三种课表视图。
- 增加表格分页、条件查询、批量停用。
- 增加 Excel 导入导出，优先支持学生、教师、课程和成绩。
- 增加 401、404 页面和更清晰的空状态提示。

## 第三阶段：扩展亮点

- 通知公告：管理员发布通知，教师和学生查看。
- 学业预警：根据不及格课程数、平均分或待修学分生成预警。
- 考试安排：维护考试时间、地点和监考教师。
- 轻量权限配置：增加菜单权限配置，但不引入复杂 RBAC 表结构。
